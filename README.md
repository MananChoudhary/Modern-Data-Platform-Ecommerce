# Modern Data Platform - Ecommerce

## Overview

This project demonstrates the design and implementation of a modern end-to-end data platform for an e-commerce business using Databricks and Azure.

The platform will process batch, incremental, CDC, and streaming data while implementing Medallion Architecture (Bronze, Silver, Gold), Delta Lake, Spark Structured Streaming, and dimensional modeling.

## Technology Stack

* Azure Databricks
* PySpark
* Delta Lake
* Spark Structured Streaming
* Python
* SQL
* Git & GitHub
